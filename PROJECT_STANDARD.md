# Project Standard: Colab + Public GitHub + Google Drive

This repository follows my standard workflow for coding projects.

## Core Workflow

This project uses:

- **Public GitHub** for code only
- **Google Colab** for running notebooks
- **Google Drive** for private or real data files
- **Codex / coding agents** for editing code and notebooks through GitHub

The repository should be safe to keep public.

Do not commit private data, real experimental data, confidential lab data, unpublished results, or large raw files to GitHub.

---

## Repository Role

GitHub is used for:

- Colab notebooks
- Python helper scripts
- README and documentation
- Example code
- Small fake/demo data only, if needed
- Requirements files
- Plotting, analysis, fitting, and utility functions

GitHub is **not** used for:

- Real experimental data
- Private research data
- Lab-confidential data
- Large CSV/XLSX/DAT/TXT data files
- Personal files
- API keys, tokens, passwords, or secrets

---

## Google Drive Role

Google Drive is used for private data.

Real data should be stored in Google Drive, for example:
